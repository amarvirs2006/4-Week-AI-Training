## Introduction to Python

Python is a high-level, interpreted programming language known for its simple and readable syntax. It is widely used in web development, data science, automation, artificial intelligence, and many other fields.

## Features of Python

* Easy to learn and use
* Readable syntax
* Open-source and free
* Platform independent
* Large collection of built-in libraries

## Variables

Variables are used to store data values.

### Example

```python
name = "John"
age = 21

print(name)
print(age)
```

## Data Types

Data types specify the kind of value a variable can store.

### Numeric Data Types

Python provides three main numeric data types:

* **int** – Stores whole numbers.
* **float** – Stores decimal numbers.
* **complex** – Stores complex numbers.

#### Example

```python
age = 21          # int
height = 5.9      # float
number = 2 + 3j   # complex
```

### Boolean Data Type

The Boolean data type represents logical values and can have only two possible values:

* `True`
* `False`

#### Example

```python
is_student = True
has_license = False
```

## Python Keywords

Keywords are reserved words in Python that have special meanings and cannot be used as variable names.

Some examples are:

* `if`
* `else`
* `for`
* `while`
* `break`
* `continue`
* `def`
* `return`

## Operators

Operators are used to perform operations on values and variables.

### Arithmetic Operators

`+`, `-`, `*`, `/`, `%`

### Comparison Operators

`==`, `!=`, `>`, `<`, `>=`, `<=`

### Logical Operators

`and`, `or`, `not`

## Basic Output

The `print()` function is used to display output.

```python
print("Hello, Python!")
```
