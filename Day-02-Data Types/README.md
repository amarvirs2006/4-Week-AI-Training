# Day 2 – Python Collections

Today, I learned about Python's built-in collection data types: Lists, Tuples, Dictionaries, and Sets. These data structures help store and organize multiple values efficiently.

## Topics Learned

* Lists
* Tuples
* Dictionaries
* Sets
* Common Functions of Each Collection

---

## Lists

Lists are ordered, mutable (changeable) collections that allow duplicate values.

### Example

```python
fruits = ["apple", "banana", "mango"]
```

### Common List Functions

* `append()` – Adds an element at the end.
* `insert()` – Inserts an element at a specific position.
* `remove()` – Removes a specified element.
* `pop()` – Removes an element using its index.
* `sort()` – Sorts the list.
* `reverse()` – Reverses the list.
* `count()` – Counts occurrences of a value.
* `index()` – Returns the index of an element.
* `copy()` – Creates a copy of the list.
* `clear()` – Removes all elements.

---

## Tuples

Tuples are ordered and immutable collections.

### Example

```python
numbers = (10, 20, 30, 40)
```

### Common Tuple Functions

* `count()` – Counts occurrences of a value.
* `index()` – Returns the index of an element.

---

## Dictionaries

Dictionaries store data in key-value pairs.

### Example

```python
student = {
    "name": "John",
    "age": 21
}
```

### Common Dictionary Functions

* `get()` – Returns the value of a key.
* `keys()` – Returns all keys.
* `values()` – Returns all values.
* `items()` – Returns key-value pairs.
* `update()` – Updates the dictionary.
* `pop()` – Removes a specified key.
* `popitem()` – Removes the last inserted item.
* `copy()` – Creates a copy of the dictionary.
* `clear()` – Removes all items.

---

## Sets

Sets are unordered collections that do not allow duplicate values.

### Example

```python
colors = {"red", "blue", "green"}
```

### Common Set Functions

* `add()`
* `remove()`
* `pop()`
* `discard()`
* `copy()`
* `issuperset()`
* `issubset()`
* `update()`
* `union()`
* `intersection()`
* `difference()`

---

## What I Learned Today

* Lists are mutable and ordered.
* Tuples are immutable.
* Dictionaries store information using key-value pairs.
* Sets automatically remove duplicate values.
* Each collection provides useful built-in methods to manipulate data.
